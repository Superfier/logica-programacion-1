# 🧩 Lógica de Programación 1 – JavaScript

Este proyecto es un ejercicio práctico de **lógica de programación** utilizando **JavaScript**.  
El objetivo es reforzar conceptos básicos como entrada de datos, comparación, ordenamiento y salida de resultados.

---

## 📌 Descripción del programa

El programa realiza lo siguiente:

1. Solicita al usuario **3 números** mediante `prompt`.
2. Convierte los valores ingresados a números 
3. Analiza los números para identificar:
   - El **mayor**
   - El **del centro**
   - El **menor**
4. Ordena los números:
   - De **menor a mayor**
   - De **mayor a menor**
5. Verifica si los números son **iguales** y muestra un mensaje especial en ese caso.
6. Imprime los resultados en **consola** (puede adaptarse para mostrar en el DOM).
